## Experiment Log

Record each experiment for reproducibility:
- Dataset + split details
- Config (link to file under `configs/`)
- Model type + hyperparameters
- Metrics (accuracy, precision/recall/F1, ROC/PR, FP/FN notes)
- Artifacts path
- Observations / next actions
